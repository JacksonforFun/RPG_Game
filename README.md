# 🎭 Сказочник — Мультиплеер RPG

Пошаговая текстовая RPG в стиле Jackbox с настоящим мультиплеером.
Ваш ПК — сервер. AI-ведущий (LLM) оценивает действия. ComfyUI рисует иллюстрации.

---

## 🚀 Быстрый старт (3 шага)

### 1. Установка

```bash
# Клиент
npm install

# Сервер
cd server && npm install && cd ..
```

### 2. Запуск сервера

```bash
cd server
npm start
```

> Сервер: `http://localhost:3001`

### 3. Запуск клиента

```bash
npm run dev
```

> Клиент: `http://localhost:5173`

**Готово!** Откройте клиент в браузере → создайте комнату → игроки подключаются по коду.

## 🌐 Игроки из других городов (ngrok)

## 🤖 AI-ведущий (Ollama)

```bash
# Установите Ollama: https://ollama.ai
ollama pull nous-hermes2
```

В настройках игры (⚙️):
- **LLM URL**: `http://localhost:11434`
- **Модель**: `nous-hermes2`

### Как AI оценивает действия

AI смотрит на **осмысленность** действия:

| Действие | Feasibility | Relevance | DC |
|----------|-------------|-----------|-----|
| "Атакую гоблина мечом" | 80% | 90% | 12 |
| "Танцую" во время боя | 60% | 10% | 22 |
| "Летаю" (без магии) | 5% | 20% | 28 |

Бессмысленные действия → высокий DC → нужен крит.

## 🎮 Как играть

1. **Хост**: создаёт комнату, видит код (например `AB12`)
2. **Игроки**: открывают URL → вводят код → создают персонажа
3. **Хост**: нажимает «Начать экспедицию»
4. **Каждый ход** (60-90 сек):
   - Игроки пишут действия текстом
   - AI-мастер оценивает и бросает d20
   - Все видят результат + картинку
5. Игра длится 12 ходов

## ⚔️ Для хоста: редактор контента

В панели управления хоста (вкладка «Контент») можно создавать:

- **Классы** — с уникальными статами и описанием для AI
- **Расы** — с модификаторами и способностями
- **Предметы** — оружие, зелья, артефакты (с прочностью/зарядами)
- **Баффы/дебаффы** — яды, благословения, увечья

> ⚠️ **Важно**: поле «Описание для AI» влияет на оценку действий!

## 🐛 Проблемы

| Проблема | Решение |
|----------|---------|
| Игроки не подключаются | Проверьте `npm start` в папке server |
| LLM не отвечает | Проверьте `ollama list` — модель скачана? |
| ComfyUI не работает | Запустите с `--enable-cors-header` |
| ngrok не работает | Авторизуйтесь: `ngrok config add-authtoken XXX` |
